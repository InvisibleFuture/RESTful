# ✨ RESTful
RESTful WEB

* 对象都是数据, 管理对象的 object 接管所有对象访问
* 对象的版本摘要信息, 同索引 都不是对象本身
* 但摘要信息需要实时更新
* 对象仅在触发某个状态时高频分发而进入内存
* 对象修改是日志事件, 只有增量, 某个时机合并日志
